# FARPHA Digital Twin Foundation V1

## Objetivo

Criar a primeira área dedicada ao gémeo digital da exploração, reutilizando os dados reais de explorações, talhões e mapa GIS.

## Incluído

- Página `/digital-twin`.
- Camadas operacionais configuráveis.
- Mapa vivo com explorações e talhões reais.
- Índice Living Farm preliminar.
- Linha temporal preparada para dados históricos.
- Estrutura preparada para sensores, máquinas, fotografias, biodiversidade e satélite.

## Limitações desta versão

- As camadas futuras aparecem desativadas até existirem integrações e dados.
- A linha temporal ainda não filtra os dados reais.
- O Living Farm Score é um índice interno preliminar, não uma certificação.
