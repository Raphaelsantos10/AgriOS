# 🌾 AgriOS

## Smart Farming Platform

AgriOS é uma plataforma inteligente de gestão agrícola construída para produtores de toda a Europa.

## Objetivos

- Gestão de explorações
- Agricultura de precisão
- Inteligência Artificial
- Drones
- Satélites
- Digital Twin
- Centro de Operações

## Tecnologias

- React
- TypeScript
- Vite
- Supabase
- PostgreSQL
- Tailwind CSS
- MapLibre
- OpenAI
