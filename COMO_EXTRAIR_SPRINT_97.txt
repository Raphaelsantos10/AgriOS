FARPHA - SPRINT 97
CONTA E CONFIGURACOES DO UTILIZADOR

1. Baixe FARPHA_Sprint_97_Conta_Configuracoes.zip para a pasta Downloads.
2. Feche o FARPHA e o VS Code.
3. Abra um novo Git Bash.
4. Execute os comandos abaixo, um de cada vez:

taskkill //F //IM node.exe 2>/dev/null || true
cd ~/Documents/Projetos
BACKUP="AgriOS_backup_sprint96_$(date +%Y%m%d_%H%M%S)"
TEMP="FARPHA_sprint97_temp_$(date +%Y%m%d_%H%M%S)"
cp -a AgriOS "$BACKUP"
mkdir "$TEMP"
unzip -o ~/Downloads/FARPHA_Sprint_97_Conta_Configuracoes.zip -d "$TEMP"
cp -a "$TEMP/AgriOS/." AgriOS/
cd ~/Documents/Projetos/AgriOS/frontend
npm install
npm run audit:prod
npm run validate
npm run dev

PUBLICAR NO GITHUB

cd ~/Documents/Projetos/AgriOS
git status
git add .
git commit -m "feat: Sprint 97 - conta e configuracoes do utilizador"
git pull --rebase origin main
git push origin main
