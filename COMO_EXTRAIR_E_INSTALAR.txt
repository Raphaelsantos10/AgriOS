ATIVAR GOOGLE (opcional):

1. Configure uma aplicação OAuth no Google Cloud.
2. No Supabase: Authentication > Sign In / Providers > Google.
3. Copie para o Google a Callback URL mostrada pelo Supabase.
4. Coloque Client ID e Client Secret no fornecedor Google do Supabase e guarde.
5. No Supabase: Authentication > URL Configuration. Adicione o endereço do FARPHA às Redirect URLs.
6. No Git Bash: nano ~/Documents/Projetos/AgriOS/frontend/.env.local
7. Acrescente: VITE_GOOGLE_AUTH_ENABLED=true
8. Guarde com Ctrl+O, Enter, Ctrl+X e reinicie npm run dev.

ATIVAR MICROSOFT (opcional):

1. Registe uma aplicação no Microsoft Entra ID.
2. No Supabase: Authentication > Sign In / Providers > Azure (Microsoft).
3. Configure a Callback URL, Client ID e Client Secret indicados.
4. Acrescente a frontend/.env.local: VITE_MICROSOFT_AUTH_ENABLED=true
5. Reinicie npm run dev.

IMPORTANTE: contas Google/Microsoft novas são criadas como “Consulta” e inativas.
O proprietário deve ativá-las no SQL Editor:

update public.user_profiles set active = true, role = 'viewer'
where id = (select id from auth.users where email = 'EMAIL_DO_UTILIZADOR');
