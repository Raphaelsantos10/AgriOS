FARPHA — SPRINT 86 — PORTAIS OFICIAIS CONTEXTUAIS

Feche VS Code e terminais dentro de AgriOS. Cole este bloco completo no Git Bash:

cd ~
taskkill //F //IM node.exe 2>/dev/null || true
cd ~/Documents/Projetos
BACKUP="AgriOS_backup_sprint85_$(date +%Y%m%d_%H%M%S)"
mv AgriOS "$BACKUP"
unzip -o ~/Downloads/FARPHA_Sprint_86_Portais_Oficiais_Contextuais.zip
cp "$BACKUP/frontend/.env.local" ~/Documents/Projetos/AgriOS/frontend/.env.local
cp -r "$BACKUP/.git" ~/Documents/Projetos/AgriOS/.git
cd ~/Documents/Projetos/AgriOS/frontend
npm install
npm run audit:prod
npm run validate
npm run preview

Esperado: zero vulnerabilidades e 151 testes aprovados. Teste em Sistema > Assistente de Auditoria. Os portais aparecem apenas nos itens com Atenção. Para parar, pressione Ctrl+C.
