# FARPHA — Sprint Enterprise 01

Entrega focada em UX, Design System e Dashboard, com princípios de Don Norman:

- Visibilidade do estado do sistema e sincronização.
- Pesquisa global com atalho Ctrl/Cmd + K.
- Tema claro/escuro persistente.
- Estado recolhido da sidebar persistente.
- Navegação por teclado e link “Saltar para o conteúdo”.
- Melhor semântica e rótulos ARIA.
- Respeito por redução de movimento.
- Dashboard com hierarquia visual mais clara.
- Novo painel “O que precisa da sua atenção”.
- Ações prioritárias com linguagem clara, contexto e urgência.
- Melhor contraste, espaçamento e consistência de componentes.
- Build de produção validado em 17/07/2026.

## Instalação

```bash
npm install
npm run dev
```

## Validação

```bash
npm run build
```
