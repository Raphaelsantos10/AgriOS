# Sprint 84 — Assistente de Auditoria

- Tela central no menu Sistema que lê os registos locais já existentes.
- Oito áreas: perfil, obrigações, documentos, tratamentos, água, IFAP/PEPAC, fiscal/laboral/seguros e rastreabilidade.
- Classificação Preparado, Atenção ou Não aplicável.
- Índice indicativo calculado somente sobre as áreas aplicáveis.
- Explicação da pendência e ligação direta para o módulo de origem.
- Nenhum registo é duplicado, corrigido ou submetido automaticamente.
- Três novos testes; total acumulado de 145.

O resultado não constitui auditoria oficial, certificação ou parecer de conformidade. Depende dos dados inseridos no navegador e deve ser confrontado com documentos originais, regras atuais, autoridades e profissionais competentes.
