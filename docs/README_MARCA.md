# FARPHA — Pacote Mestre de Marca Transparente

Este pacote preserva o desenho oficial enviado pelo criador da marca.

## Ficheiros principais

- `farpha-logo-transparent-master.png` — logótipo completo com transparência.
- `farpha-symbol-transparent-master.png` — símbolo isolado com transparência.
- Versões `2x` e `4x` — para impressão, apresentações e ecrãs de alta densidade.
- Ícones de 16 a 512 px — favicon, PWA, Android e Apple Touch Icon.
- Previews em fundo escuro e claro — apenas para visualização.

## Integração no projeto

Copiar a estrutura `frontend/` para a raiz do projeto `AgriOS/`.

Caminhos finais:

- `frontend/src/assets/brand/farpha-logo.png`
- `frontend/src/assets/brand/farpha-symbol.png`
- `frontend/public/favicon-farpha.png`
- `frontend/public/apple-touch-icon.png`
- `frontend/public/icon-192.png`
- `frontend/public/icon-512.png`

## Regra de uso

Não deformar, inclinar, recolorir ou alterar proporções. Manter margem livre em volta do símbolo e do nome.
