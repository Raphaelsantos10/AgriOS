# FARPHA 2.0 Foundation — Fase 1

## Incluído
- Event Bus tipado;
- Feature Flags locais;
- tokens oficiais da identidade FARPHA;
- componentes-base: Button, Card, Badge e Input;
- variáveis CSS para tema claro e futuro tema escuro;
- ADRs de arquitetura.

## Impacto
Esta fase não altera visualmente as páginas existentes e não muda o Supabase. Ela cria a base para migrar cada módulo sem quebrar funcionalidades.

## Próxima fase
Migrar primeiro o Layout, Sidebar e Header para o Design System FARPHA, mantendo as rotas e features atuais.
