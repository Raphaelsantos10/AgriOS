# FARPHA Foundation — Fase 4

## Sidebar premium e navegação visual

Esta entrega consolida a identidade visual oficial no menu principal sem alterar a lógica dos módulos.

### Alterações

- uso dos assets transparentes oficiais FARPHA;
- logótipo maior e mais legível;
- agrupamento em Operações, Inteligência, Monitorização e Sistema;
- melhor espaçamento entre itens;
- estado ativo com barra lateral e gradiente FARPHA;
- ícones ampliados;
- estados “Breve” mais discretos;
- cartão de utilizador com estado online;
- versão recolhida otimizada.

### Validação

Executar:

```bash
npm run lint
npm run build
npm run dev
```
