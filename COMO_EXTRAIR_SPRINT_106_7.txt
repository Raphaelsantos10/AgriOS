FARPHA Sprint 106.7 — instalação no Git Bash

cd ~
taskkill //F //IM node.exe 2>/dev/null || true
cd ~/Documents/Projetos
BACKUP="AgriOS_backup_sprint106_6_$(date +%Y%m%d_%H%M%S)"
mv AgriOS "$BACKUP"
unzip -o ~/Downloads/FARPHA_Sprint_106_7_Cadastro_Onboarding.zip -d ~/Documents/Projetos
cp "$BACKUP/frontend/.env.local" ~/Documents/Projetos/AgriOS/frontend/.env.local 2>/dev/null || true
cp -r "$BACKUP/.git" ~/Documents/Projetos/AgriOS/.git
cd ~/Documents/Projetos/AgriOS/frontend
npm install
rm -rf node_modules/.vite
npm run validate
npm run dev

Supabase obrigatório

1. Abra o SQL Editor.
2. Execute database/SPRINT_106_7_SELF_SERVICE_AUTH.sql.
3. Em Authentication > Providers configure Email, Google e Azure/Microsoft.
4. Em URL Configuration adicione http://localhost:5173 e a URL pública.
