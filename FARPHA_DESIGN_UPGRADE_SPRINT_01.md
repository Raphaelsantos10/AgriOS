# FARPHA — Design Upgrade Sprint 01

## Melhorias entregues
- Design tokens profissionais revistos
- Tema claro e escuro funcional com persistência local
- Header Enterprise com pesquisa, estado online, tema, notificações e perfil
- Layout principal com melhor contraste, espaçamento e hierarquia
- Dashboard reconstruído em superfícies claras e consistentes
- KPIs, alertas, cards operacionais e painel de IA redesenhados
- Sidebar refinada com navegação ativa mais discreta e profissional
- Acessibilidade: foco visível e suporte a redução de movimento
- Build de produção validado com sucesso

## Como usar
1. Extraia o ZIP.
2. Substitua a pasta frontend atual pela pasta frontend deste pacote.
3. Execute `npm install`.
4. Execute `npm run dev`.

## Validação
`npm run build` concluído com sucesso.
